## Requirements

To use library you need to have [node](https://nodejs.org) and [npm](https://www.npmjs.com) installed in your machine:

-   node `6.0+`
-   npm `3.0+`

## Installation

To install the library run following command

```bash
  npm i --save myrmidon
```
